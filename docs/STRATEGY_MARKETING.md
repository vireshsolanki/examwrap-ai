# ExamWrapAI - Strategy & Marketing Playbook 🚀

## 1. The Competitive Landscape 🕵️‍♂️
You asked if there are other apps that create questions from PDFs. The answer is **yes**, but most are generic.
*   **The Big Players**: Humata.ai, ChatPDF, and Quizgecko.
*   **Your Advantage**: They are "Generalists." You are an **"Exam Expert."**
*   **The Gap**: Most competitors give you a quiz, but they don't give you a **7-Day Adaptive Revision Plan** or **Grounding Citations** like we built. 
*   **Positioning**: Don't call it a "PDF Quizzer." Call it an **"AI Mock Exam Architect."**

## 2. Essential Feature Roadmap (To Win the Market) 🗺️
To pitch to institutes like PhysicsWallah or Allen, you need these:

### A. 📸 The Diagram Problem (OCR/Multimodal)
Competitive exams (JEE, NEET, CAT) are 50% diagrams and equations. 
*   **Action**: Use Gemini's vision capability to "see" diagrams and generate questions about them. A chemistry tool that can't read a benzene ring is a no-go for big institutes.

### B. 📊 Institutional Dashboard
Institutes don't care about one student; they care about 10,000.
*   **Action**: Build a view that shows "Topic-wise Accuracy for the whole class."

### C. ⏳ Simulated Exam Environment
*   **Action**: Create a mode that mimics the exact UI of the actual exam (e.g., the NTA interface for JEE). Students pay for "Exam Temperament."

## 3. The Technical Scale-Up ⚙️
### Vector DB (Milvus/Pinecone)
*   **Why**: You mentioned the 10MB limit. RAG (Retrieval Augmented Generation) using a Vector DB is how you handle 500-page textbooks without hitting the AI context limit or paying 10x in tokens.

## 4. Reddit Launch Playbook (Phase 1) 📣
**Target Subreddits**: `r/GetStudying`, `r/EngineeringStudents`, `r/medicalschool`, `r/productivity`.

### The Strategy:
1.  **Don't Spam**: If you post a link, you'll be banned. 
2.  **The "Problem/Solution" post**: "I was frustrated that I couldn't test myself on my own class notes, so I built a tool in Rust that generates mock exams from PDFs. It's private (local-only) and gives you a 7-day study plan."
3.  **The "Giveaway"**: Offer to run a PDF for the first 50 people who comment.
4.  **The Feedback Loop**: Use Reddit to find "Pain Points" you didn't think of.

## 5. Pitching to Big Institutes (Phase 2) 💼
When you talk to companies like Akash or Allen, don't sell "AI." Sell **"Cost Reduction" and "Efficiency."**

### The Pitch:
*   "Your teachers spend 20 hours a week creating practice tests. My tool does it in 20 seconds with 99% accuracy and source verification."
*   "We provide every student with a Personalized Revision Scheduler—something a human teacher cannot do for 5,000 students at once."
